O que são Variáveis?

Em Python, variáveis são usadas para armazenar dados. Elas são criadas automaticamente quando você atribui um valor a um nome. Não é necessário declarar o tipo da variável, pois o Python faz isso automaticamente.

Por exemplo:

nome = "Maria"

Aqui, criamos uma variável chamada nome que guarda o texto "Maria".

Exemplos de Variáveis

📌 Número Inteiro

idade = 25

📌 Texto (String)

cidade = "São Paulo"

📌 Número Decimal (Float)

altura = 1.75

📌 Booleano

aprovado = True
