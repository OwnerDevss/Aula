O que são Processos?

Processos são instâncias de programas em execução. Eles são fundamentais para que o sistema operacional gerencie o uso de recursos.

Estados dos Processos

  • Pronto: O processo está aguardando para ser executado.
  • Executando: O processo está sendo processado pela CPU.
  • Bloqueado: O processo está aguardando por um recurso (exemplo: entrada do usuário).

Exemplo:

Um navegador da web é um processo que pode estar "executando" ao carregar uma página, "bloqueado" ao esperar por uma resposta do servidor, ou "pronto" para processar novas solicitações.

Gerenciamento de Memória

A memória é gerenciada para garantir que vários processos possam coexistir eficientemente, sem interferir uns nos outros.

Técnicas de Gerenciamento

  • Particionamento: Divide a memória em blocos fixos ou dinâmicos para alocação.
  • Paginação: Divide a memória em páginas menores e organiza os dados em blocos.
  • Memória Virtual: Usa o disco rígido para estender a memória principal, permitindo que mais processos sejam executados.

Exemplo:

Ao abrir vários aplicativos no computador, o sistema operacional usa a memória virtual para manter os aplicativos ativos mesmo que a memória RAM esteja cheia.

Gerenciamento de Arquivos

O gerenciamento de arquivos organiza e controla como os dados são armazenados, acessados e manipulados no sistema.

Sistemas de Arquivos

  • FAT32: Estrutura simples, usada em dispositivos antigos.
  • NTFS: Robusto, com suporte a permissões e recuperação de falhas.
  • EXT4: Um sistema de arquivos moderno usado em sistemas Linux.

Exemplo:

Ao salvar um documento no Word, ele é armazenado no disco rígido usando um sistema de arquivos (ex.: NTFS), que organiza as informações em blocos físicos.
